# คู่มือการติดตั้งและใช้งาน (Installation & User Manual)

## 1. รายละเอียดโครงงาน (Project Description)
โปรแกรม Twitch Bad Word Detector เป็นแอปพลิเคชัน Desktop (GUI) ที่พัฒนาด้วยภาษา Python สำหรับตรวจจับและแจ้งเตือนคำหยาบในช่องแชท Twitch แบบ Real-time รองรับทั้งภาษาไทยและภาษาอังกฤษ โดยมีการแสดงผลสถิติผ่าน Dashboard และแจ้งเตือนด้วยเสียงเมื่อพบคำหยาบ

## 2. สิ่งที่ต้องมี (Prerequisites)
ก่อนเริ่มใช้งาน โปรดตรวจสอบว่าเครื่องคอมพิวเตอร์ของคุณมีสิ่งเหล่านี้:
* **Python 3.8 หรือใหม่กว่า** (แนะนำ Python 3.10 ขึ้นไป)
* การเชื่อมต่ออินเทอร์เน็ต (สำหรับการเชื่อมต่อ Twitch API)
* บัญชี GitHub (สำหรับการ Clone project)

## 3. วิธีการติดตั้งและรันโปรแกรม (Installation & Execution)

### วิธีที่ 1: รันแบบง่าย (แนะนำ)
เพื่อให้สะดวกต่อการใช้งาน ได้มีการจัดทำไฟล์สำหรับรันอัตโนมัติไว้ให้แล้ว
1.  ดาวน์โหลดหรือ Clone repository นี้ลงในเครื่อง
2.  ดับเบิ้ลคลิกที่ไฟล์ **`run.bat`**
3.  ระบบจะทำการติดตั้ง Library ที่จำเป็นและเปิดโปรแกรมให้ทันที

### วิธีที่ 2: รันผ่าน Command Line (Manual)
หากต้องการติดตั้งด้วยตนเอง ให้ทำตามขั้นตอนดังนี้:

1. เปิด Terminal หรือ Command Prompt ที่โฟลเดอร์ของโปรเจค
2. ติดตั้ง Dependencies:
   pip install -r requirements.txt
3. รันโปรแกรม :
   python main_gui.py

## 4. วิธีการใช้งาน (Usage Guide)
1. Start Program: เมื่อเปิดโปรแกรมขึ้นมา จะพบหน้าต่างหลัก
2. Enter Channel: กรอกชื่อช่อง Twitch ที่ต้องการตรวจสอบในช่อง "Channel Name" (ไม่ต้องใส่ #)
3. Connect: กดปุ่ม "เชื่อมต่อ Twitch" เพื่อเริ่มการทำงาน
    - สถานะจะเปลี่ยนเป็น "Connected"
    - ข้อความแชทจะเริ่มแสดงในหน้าต่าง "ข้อความแชท"
    - หากพบคำหยาบ ระบบจะแจ้งเตือนและบันทึกในแท็บ "คำหยาบที่พบ"
4. Dashboard: กดปุ่ม "เปิด Dashboard" เพื่อดูสถิติการตรวจจับแบบกราฟิกและ Real-time
5. Stop: กดปุ่ม "ยกเลิกการเชื่อมต่อ" เมื่อต้องการหยุดการทำงาน
6. Export Log: สามารถกดปุ่ม Export Log เพื่อบันทึกประวัติการตรวจจับเป็นไฟล์ CSV ได้
